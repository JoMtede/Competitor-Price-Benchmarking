# Competitor-Price-Benchmarking
Data Cabinet Pricing & Competitor Benchmarking Analysis
Project Overview
This project analyzes Africa and Asia data cabinet pricing across multiple competitors and countries using R for data cleaning, statistical analysis, regression modelling, and market segmentation, with interactive dashboards created in Tableau.
The goal of the project was to:
Benchmark competitor pricing
Identify pricing outliers
Analyze country-level pricing trends
Build a predictive pricing model
Segment products into premium, mid-market, and budget categories
Create interactive business intelligence dashboards
